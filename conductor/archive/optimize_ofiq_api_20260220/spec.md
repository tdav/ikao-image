# Track Specification: Optimization of OFIQ API

## Description
This track focuses on optimizing the existing C# / C++ bridge for image processing and extending the API to expose more quality measures from the OFIQ library.

## Goals
- Enhance image data transfer performance between C# and C++.
- Add support for remaining quality measures in NativeInvoke.cs.
- Improve error handling for edge-case images.

## Requirements
- Maintain compatibility with ISO/IEC 29794-5.
- Code test coverage for new API must be >= 80%.
